Developer API Documentation

Integrate BongoDB into any website or application using simple HTTP requests.

🚀 Quick Start

BongoDB runs locally on http://localhost:5001. You can send JSON data to your cloud using these endpoints.

1. Insert Record

POST /api/v1/insert 
{
  "collection": "Products",
  "data": {
    "name": "Luxury Watch",
    "price": 299,
    "stock": 15
  }
}

2. Find Records

GET /api/v1/find?collection=Products

Returns all rows from the specified Google Sheet collection.

Cloud Storage

To upload images, use the /api/v1/upload endpoint. Files are stored directly in your linked Google Drive folder.

Security

BongoDB uses your local OAuth session. No API Key is required when running locally alongside your development server.
